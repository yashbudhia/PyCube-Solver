User Guide

Basic Controls

  • Generate Scramble: Randomly scrambles the cube
  • Solve Cube: Finds optimal CFOP solution
  • Reset: Returns cube to solved state
  • Apply Moves: Execute custom move sequences
  • Animate Solution: Visual step-by-step animation

CFOP Method

  • Cross: Solve white cross on bottom
  • F2L: First Two Layers simultaneously
  • OLL: Orient Last Layer (yellow face)
  • PLL: Permute Last Layer (final solve)

📚 Move Notation Guide

Basic Moves
R Right face clockwise
R' Right face counter-clockwise
L Left face clockwise
L' Left face counter-clockwise
U Upper face clockwise
U' Upper face counter-clockwise
D Down face clockwise
D' Down face counter-clockwise
F Front face clockwise
F' Front face counter-clockwise
B Back face clockwise
B' Back face counter-clockwise
Advanced Notation
R2 Right face 180° turn
Rw Right two layers clockwise
M Middle layer (like L)
x Rotate entire cube like R
y Rotate entire cube like U
z Rotate entire cube like F
CFOP Method Steps
Cross Form a white cross on the bottom face
F2L First Two Layers - Complete bottom two layers
OLL Orientation of Last Layer - Make top face yellow
PLL Permutation of Last Layer - Position final pieces

Tips

  • Start with shorter scrambles (10-15 moves)
  • Watch the animation to learn algorithms
  • Practice with custom move sequences
  • Use the 2D display to analyze cube state
